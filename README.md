Project Description
This project demonstrates how to build a RESTful API that allows users to add, view, update, and delete video records.  
It is built using Node.js, Express.js, and MongoDB using Mongoose as the ORM.

This assignment was prepared as part of the APIs in Action Bootcamp, focusing on:
- Express Routing
- MongoDB Database Integration
- Schema Validation
- CRUD API Endpoints
